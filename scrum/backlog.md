## Criterios de aceite finais

- Firmware compila na Arduino IDE com as bibliotecas indicadas.
- O ESP32 publica telemetria nos topicos definidos.
- O ESP32 recebe comandos MQTT de buzzer e reset.
- O script Python insere mensagens recebidas no MySQL.
- A documentacao descreve o prototipo sem caracterizar o projeto como dispositivo medico certificado.
- Evidencias reais sao adicionadas somente apos testes praticos.
